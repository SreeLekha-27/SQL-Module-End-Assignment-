# E-Learning Platform Purchase Analysis 

##  Project Overview  
This project analyzes an e-learning platform's purchase data to understand user behavior, course popularity, and revenue trends. The analysis is done using SQL to answer key business questions and support data-driven decisions.

---

##  Dataset Description  
The dataset contains information about:  
- Users  
- Courses  
- Purchases  
- Payment details  
- Enrollment dates  

---

## Business Objectives  
- Identify top-selling courses  
- Analyze user purchase behavior  
- Calculate total revenue and monthly trends  
- Find repeat customers and high-value users  
- Understand category-wise course demand  

---

## Key SQL Queries Used  
- SELECT, WHERE, GROUP BY, HAVING  
- INNER JOIN, LEFT JOIN  
- Subqueries  
- Aggregate functions (SUM, COUNT, AVG)  
- Date functions  

---

## Key Insights  
- Identified top-performing courses and categories  
- Found monthly revenue growth trends  
- Discovered user purchasing patterns and repeat customers  
- Highlighted high-revenue customers and popular platforms  

---

## How to Run the Project  
1. Import the dataset into MySQL / PostgreSQL  
2. Run the provided SQL queries  
3. Analyze results using SQL client or export to Excel/Power BI  

---

## Conclusion  
This project demonstrates practical SQL skills for business analysis, including joins, aggregations, and real-world problem solving. It showcases how SQL can be used to extract insights and support decision-making in digital platforms.
